# vacation_manager

## Requirements

```
python: >3.7, <3.11
```

## Running

### 1 Clone the repository

```bash
git clone https://github.com/prateekb1/vacation_manager.git
```

### 2 Installation

```bash
pip install -r requirements.txt
```

### 3 Create Virtual Environment

```bash
pipenv shell
```

### 4 Running on server

```bash
python manage.py runserver
```
